Installation
============

::

   $ go get -u github.com/sampoapp/sampo-cli/sampo

   $ git clone https://github.com/sampoapp/sampo-demos.git

Usage
=====

::

   $ cd sampo-demos

   $ sampo build batman

   $ sampo build matrix
